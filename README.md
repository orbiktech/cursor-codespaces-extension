# Cursor Codespaces

A VS Code extension that provides seamless GitHub Codespaces connectivity via GitHub CLI.

## Description

Cursor Codespaces enables developers to easily connect to and manage GitHub Codespaces directly from VS Code, leveraging the GitHub CLI for a streamlined workflow.

## Features

- Connect to GitHub Codespaces
- Manage Codespaces instances
- Seamless integration with GitHub CLI

## Requirements

- GitHub CLI (`gh`) must be installed and authenticated
- VS Code version 1.106.1 or higher

## Extension Settings

This extension currently provides basic Codespaces connectivity functionality.
